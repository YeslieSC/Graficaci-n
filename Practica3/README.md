# PrActica 3 - Archivo Numerico

## Objetivo
Leer el contendio de un archivo para dibujar una imagen con turtle

## Descripción del proyecto
Este proyecto genera una matriz de numeros aleatorios entre 0 y 9
la guarda en un archivo de texto y la representa graficamente utilizando la libreria `turtle`
cada numero se traduce en un color especifico, formando una cuadricula visual de 100x100 pixeles
se dibuja la matriz en pantalla, donde cada numero se representa como un cuadrado de color especifico

## Instrucciones para ejecutar el proyecto
1 Abrir el proyecto en Visual Studio o Visual Studio Code
2 Tener Python instalado
3 Ejecutar el archivo principal practica3_archivoNumerico.py
Se abrira una ventana donde se dibujara la matriz de colores

## Estructura del programa
Generación de la matriz:
Se utiliza la librería `random` para generar numeros aleatorios
Se utiliza el modulo turtle para dibujar la matriz
El archivo `matriz.txt` se crea automaticamente y se sobrescribe en cada ejecucion
