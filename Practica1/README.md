# Practica 1 –  Dibujar Figuras Geometricas

## Objetivo
Utilizar la libreria turtle para dibujar figuras geometricas en Python.

# Este proyecto es compatible con python 3.10 y solo requiere el modulo turtle

## Descripcion
El programa dibuja cuatro figuras usando funciones definidas por el usuario  
Cada figura recibe parametros como posicion tamaño y color  
Las figuras se dibujan alineadas horizontalmente para mantener orden
El tamaño, color y posicion de cada figura se puede modificar al final del codigo en las llamadas a las funciones


## Instrucciones para ejecutar el proyecto
1 Abrir el proyecto en Visual Studio o Visual Studio Code
2 Tener Python instalado 
3 Ejecutar el archivo principal practica1_figurasG_.py
Se abrira una ventana donde se dibujaran las figuras 

Figuras en el proyecto:
- Cuadrado
- Triangulo
- Circulo
- Linea recta
